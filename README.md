# Modularized DDPM

## Overview
This project implements Denoising Diffusion Probabilistic Models (DDPM) on implied volatility forcasting.

## Project Structure
- **/data**: Data will be used for this project. The 00_acquire_data.py will fill this folder once is run.
- **/iv_ddpm**: The main scripts.
- **/models**: Models that can be used later.
- **/results**: The plots for further analysis.
- **/scripts**: Code to execute the main scripts.

## Installation
To set up the project, clone the repository and install the required dependencies:
```bash
git clone <repository-url>
cd modularized_DDPM
pip install -r requirements.txt
```
